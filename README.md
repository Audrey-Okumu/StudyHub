# StudyHub

A full-stack student productivity platform built while learning Android Development and Spring Boot.

## Goal

Build one real application while learning:

* Kotlin
* Jetpack Compose
* Android Architecture
* Java
* Spring Boot
* PostgreSQL
* REST APIs
* Git & GitHub

Instead of creating many tutorial projects, every concept learned is implemented directly into StudyHub.

---

## Features

### Authentication

* Register
* Login
* Profile

### Courses

* Create courses
* Manage units
* Track progress

### Tasks

* Create tasks
* Mark complete
* Organize priorities

### Study Planning

* Study sessions
* Goals
* Progress tracking

### Analytics

* Productivity insights
* Study statistics
